## Core Features:

- Transaction Input: Simple data input for income and expenses.
- Spending Visualization: Visualization of spending habits, categorized for quick understanding.
- AI Powered Tips and Alerts: Generation of tips and alerts. For example, the tool may recommend an alert if a user spends a lot on some item and hasn't recorded the purchase yet.
- Data Import: Data import to simplify your input. CSV import format.
- Local Data Storage: Privacy first; store all data locally; avoid using server side databases to enhance the safety of users' information. If the user want's it they can export the data.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6) to evoke trust and stability in financial matters.
- Background color: Light, desaturated blue (#E3F2FD) to provide a clean and calming backdrop.
- Accent color: Gentle green (#81C784) to highlight gains and positive actions.
- Font pairing: 'Poppins' (sans-serif) for headlines and shorter text chunks; 'PT Sans' (sans-serif) for longer body text.
- Clean, minimalist layout with clear data presentation.
- Simple, intuitive icons to represent different transaction categories.
- Subtle animations to acknowledge user actions, such as a transaction confirmation.
